# Phase 3 — JavaScript

**Prerequisite:** [Phase 2 — Frontend](../02-frontend/README.md)

JavaScript turns static pages into interactive experiences. This phase covers the language itself, how to use it to control the browser, and how to communicate with external services. By the end you'll be able to fetch data from an API and update the page dynamically.

## Topics

| # | Topic | What You'll Learn |
|---|-------|-------------------|
| 01 | [JS Fundamentals](01-js-fundamentals.md) | Variables, functions, arrays, objects, scope, the event loop |
| 02 | [DOM Manipulation](02-dom-manipulation.md) | Selecting elements, handling events, updating the page |
| 03 | [Async JavaScript](03-async-javascript.md) | Callbacks, promises, async/await — handling things that take time |
| 04 | [Fetch & APIs](04-fetch-and-apis.md) | Making HTTP requests, working with JSON, error handling |
| 05 | [Modern JS (ES6+)](05-modern-js.md) | Arrow functions, destructuring, spread, modules, template literals |
