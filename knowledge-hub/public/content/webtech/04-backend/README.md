# Phase 4 — Backend

**Prerequisite:** [Phase 3 — JavaScript](../03-javascript/README.md)

The backend is everything the user doesn't see: the server that handles requests, the database that stores data, the logic that enforces rules. This phase uses Node.js so you write JavaScript on both sides of the stack — one language, full picture.

## Topics

| # | Topic | What You'll Learn |
|---|-------|-------------------|
| 01 | [How Servers Work](01-how-servers-work.md) | The request/response cycle from the server's perspective, ports, middleware |
| 02 | [Node.js Basics](02-nodejs-basics.md) | The Node runtime, npm, the file system, core modules |
| 03 | [REST APIs](03-rest-apis.md) | Routes, controllers, HTTP verbs, status codes, API design |
| 04 | [Databases](04-databases.md) | SQL basics, SQLite, CRUD operations, connecting to Node |
| 05 | [Authentication](05-authentication.md) | Sessions, JWTs, password hashing, protecting routes |
