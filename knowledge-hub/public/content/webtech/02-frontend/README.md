# Phase 2 — Frontend

**Prerequisite:** [Phase 1 — Foundations](../01-foundations/README.md)

Before JavaScript can do anything useful, it needs something to work with. This phase gives it that: well-structured HTML that's predictable to select, and CSS layouts that won't collapse unexpectedly. You'll go deeper on HTML and CSS than Phase 1, with a focus on how professional developers think about markup and style.

## Topics

| # | Topic | What You'll Learn |
|---|-------|-------------------|
| 01 | [HTML Semantics](01-html-semantics.md) | Semantic elements, document outline, why structure matters |
| 02 | [CSS Layout](02-css-layout.md) | Flexbox and Grid — the two layout systems you'll use for everything |
| 03 | [Responsive Design](03-responsive-design.md) | Media queries, mobile-first approach, fluid layouts |
| 04 | [Accessibility](04-accessibility.md) | ARIA roles, keyboard navigation, screen reader compatibility |
