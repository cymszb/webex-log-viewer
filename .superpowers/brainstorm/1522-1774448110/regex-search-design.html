Regex Search + Save as Topic — Design Mockup

Three states of the enhanced search input

State 1 — Plain text mode (default)
.* button is inactive (grey). Works exactly as today — plain case-insensitive substring match.
State 2 — Regex mode, valid pattern
💾
.* button is active (blue). Input uses monospace font with blue border. 💾 Save-as-topic icon appears inside the input on the right — only when regex is valid. Clicking 💾 opens the New Topic modal with this regex pre-filled in the pattern field.
State 3 — Regex mode, invalid pattern
invalid regex Red border on input. No 💾 icon. Log table shows 0 results.
