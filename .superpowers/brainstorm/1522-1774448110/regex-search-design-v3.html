Regex Search + Save as Topic — Design Mockup v3

Three states of the enhanced search input

State 1 — Plain text mode (default)
.* button inactive (grey). Plain substring match. No "Save as Topic" inside the box.
State 2 — Regex mode, valid pattern
Blue border on the box. Monospace font. Save as Topic button sits inside the right edge of the input box — only when regex is valid.
State 3 — Regex mode, invalid pattern
invalid regex Red border. No "Save as Topic" button. 0 results shown.
